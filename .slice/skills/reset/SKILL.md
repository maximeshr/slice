# reset

Resolve model/mode from `../../policy.md`: reasoning `light`, access `read`.

## Procedure

1. Summarize the current state into a short handoff: what's done, the active scope/slice, open decisions, the next action.
2. Start the next phase from that summary in a fresh context. Carry the summary, not the transcript.

## Why

Every model degrades as context fills. `reset` keeps each phase inside the smart zone.

## Output

A short handoff summary, and a lean context for the next step.

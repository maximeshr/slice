# scope

Resolve model/mode from `../../policy.md`: reasoning `heavy`, access `plan`. Write a planning artifact only, no source.

## Procedure

1. Using `../../templates/scope.md`, write one document: problem, decisions settled in align, in/out edges, and acceptance criteria (the definition of done).
2. Keep it a contract, not a tutorial. A teammate who wasn't in the align must understand the feature from this alone.
3. Save it as the feature's epic (an issue, or `docs/scopes/<feature>.md` linked to it).

## Output

One scope doc = the epic. One per feature. It feeds `slice`.
